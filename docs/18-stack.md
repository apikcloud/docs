## Core Components
The Odoo technical stack consists of the following core components:
- **Odoo Server**: The main application server that runs the Odoo framework and handles business logic.
- **PostgreSQL**: The relational database management system used to store Odoo data.
- **Reverse Proxy**: Nginx or similar, used to handle incoming requests, SSL termination, and load balancing.
