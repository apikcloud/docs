## 1. Naming Conventions

### Rules

- **Addon names**: `apik_` prefix for all Apik-specific addons (e.g., `apik_brand`, `apik_filter`)
- **File headers**: All files include copyright header: `# Copyright 2025 apik (https://apik.cloud).`
